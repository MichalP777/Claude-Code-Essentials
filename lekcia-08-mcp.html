01

Čo je Model Context Protocol

Claude Code vie čítať súbory, spúšťať príkazy a upravovať kód — ale bez MCP sa nedostane k databáze, Slacku, prehliadaču ani externému API. MCP (Model Context Protocol) je otvorený štandard od Anthropic, ktorý definuje, ako môžu AI agenti bezpečne komunikovať s externými nástrojmi.

MCP server je samostatný proces, ktorý agent môže volať ako nástroj. Každý server ponúka sadu nástrojov (tools), zdrojov (resources) a výziev (prompts).

MCP nie je proprietárny — funguje s akýmkoľvek AI agentom ktorý ho implementuje. Komunita vytvorila stovky MCP serverov pre rôzne služby. Zoznam nájdeš na mcpservers.org a github.com/modelcontextprotocol/servers.

02

Pridanie MCP servera do konfigurácie

MCP servery sa konfigurujú v settings.json — globálne alebo pre konkrétny projekt. Claude Code ich automaticky spustí ako subprocess a komunikuje s nimi cez stdio alebo SSE.

pridanie MCP servera cez CLI
# Pridanie Playwright MCP (globálne)
$claude mcp add playwright npx @playwright/mcp@latest
✓ MCP server "playwright" added to global config
# Pridanie pre konkrétny projekt
$claude mcp add --scope project postgres npx @mcp/postgres
# Zoznam nainštalovaných MCP serverov
$claude mcp list
playwright npx @playwright/mcp@latest [global]
postgres npx @mcp/postgres [project]
# Overenie že server beží správne
$claude mcp test playwright
✓ playwright: connected, 24 tools available
# Odstránenie servera
$claude mcp remove playwright
📄 ~/.claude/settings.json — manuálna konfigurácia
{ "mcpServers": { // Playwright — ovládanie prehliadača "playwright": { "command": "npx", "args": ["@playwright/mcp@latest"] }, // PostgreSQL — prístup k databáze "postgres": { "command": "npx", "args": ["@mcp/postgres"], "env": { "DATABASE_URL": "postgresql://localhost/mydb" } }, // Filesystem — rozšírený prístup k súborom "filesystem": { "command": "npx", "args": ["@modelcontextprotocol/server-filesystem", "/home/user/projects"] } } }
// bezpečnosť MCP serverov

MCP servery môžu mať prístup k databázam, súborom aj sieti. Inštaluj len dôveryhodné servery — preferuj oficiálne od @modelcontextprotocol alebo overených vývojárov. Vždy nastav len minimálne potrebné oprávnenia.

03

Playwright MCP — agent ovláda prehliadač

Playwright MCP je jeden z najpopulárnejších serverov — dáva agentovi plnú kontrolu nad prehliadačom. Agent môže otvárať URL, klikať na elementy, vypĺňať formuláre, robiť screenshoty a spúšťať JavaScript.

Playwright MCP — príklady
# Inštalácia
$claude mcp add playwright npx @playwright/mcp@latest
✓ playwright: 24 tools available
# Príklad 1: E2E test
> Otvor localhost:3000, prihlás sa ako test@example.com
  s heslom "test123" a over, že dashboard sa načítal.
playwright: navigating to localhost:3000...
playwright: filling #email with test@example.com
playwright: clicking [type=submit]...
✓ Dashboard loaded. Title: "My App — Dashboard"
# Príklad 2: Screenshot a analýza UI
> Urob screenshot stránky a nájdi všetky broken links.
playwright: screenshot saved
playwright: checking 47 links...
⚠ 3 broken links found: /about, /pricing, /docs/v1
# Príklad 3: Web scraping
> Stiahni zoznam produktov z e-shopu a ulož do JSON.
✓ products.json created (128 items)
// headless vs. viditeľný prehliadač

Playwright štandardne beží v headless móde (bez okna). Pre debugovanie pridaj --headed parameter do args — uvidíš, čo agent robí v reálnom čase.

06

Vlastný MCP server — základy

MCP server môžeš napísať v Node.js alebo Pythone pomocou oficiálneho SDK. Minimálny server definuje zoznam nástrojov a handler pre každý z nich.

📄 my-mcp-server.js — minimálny MCP server v Node.js
// npm install @modelcontextprotocol/sdk import { Server } from '@modelcontextprotocol/sdk/server/index.js'; import { StdioServerTransport } from '@modelcontextprotocol/sdk/server/stdio.js'; const server = new Server( { name: 'my-server', version: '1.0.0' }, { capabilities: { tools: {} } } ); // Definícia nástroja server.setRequestHandler(ListToolsRequestSchema, async () => ({ tools: [{ name: 'get-weather', description: 'Vráti aktuálne počasie pre dané mesto', inputSchema: { type: 'object', properties: { city: { type: 'string' } } } }] })); // Handler pre nástroj server.setRequestHandler(CallToolRequestSchema, async (req) => { const city = req.params.arguments.city; const weather = await fetchWeather(city); // tvoja logika return { content: [{ type: 'text', text: weather }] }; }); const transport = new StdioServerTransport(); await server.connect(transport);
// registrácia vlastného servera

Po vytvorení vlastného servera ho zaregistruj: claude mcp add my-server node /path/to/my-mcp-server.js

// rýchly test

Chceš, aby Claude Code agent mohol automaticky otvoriť prehliadač, vyplniť formulár a overiť výsledok. Ktorý MCP server použiješ?

A GitHub MCP — pre prácu s repozitármi a PR
B Filesystem MCP — pre rozšírený prístup k súborom
C Playwright MCP — pre ovládanie prehliadača, formulárov a E2E testy
D Fetch MCP — pre HTTP volania na externé API

MCP je otvorený štandard, ktorý dáva agentovi prístup k externým nástrojom — prehliadač, databázy, GitHub, Slack a stovky ďalších. Servery sa konfigurujú v settings.json alebo cez claude mcp add. Vlastný MCP server napíšeš v Node.js alebo Pythone pomocou oficiálneho SDK.
