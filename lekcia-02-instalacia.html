Inštalácia a
prvé kroky

Nainštaluj Claude Code na svoj systém — Windows, Linux alebo VS Code — a overiš, že všetko beží správne pomocou diagnostického príkazu doctor.

// čo sa naučíš

  • Nainštalovať Claude Code cez npm na Windows (PowerShell/CMD) a Linux
  • Prihlásiť sa cez Anthropic účet alebo API kľúč
  • Overiť prostredie príkazom claude doctor
  • Pochopiť, čo doctor kontroluje a ako riešiť bežné chyby
01

Čo potrebuješ pred inštaláciou

Claude Code je CLI nástroj distribuovaný cez npm (Node Package Manager). Na jeho beh potrebuješ iba dve veci: Node.js a účet na Anthropic.

// požiadavky

Node.js 18 alebo novší  ·  npm 8+ (súčasť Node.js)  ·  Anthropic účet alebo API kľúč

Verziu Node.js overíš takto:

terminal
$node --version
v22.14.0
$npm --version
10.9.2
// pozor

Ak nemáš Node.js, stiahni si ho z nodejs.org — odporúčame LTS verziu (20 alebo 22).

02

Inštalácia podľa platformy

Vyber si svoju platformu a postupuj podľa krokov:

bash / zsh
# Globálna inštalácia cez npm
$npm install -g @anthropic-ai/claude-code
added 47 packages in 8s
# Overenie inštalácie
$claude --version
claude-code 1.x.x
// tip

Ak dostaneš chybu EACCES (permission denied), spusti npm install -g s sudo, alebo si nastav npm prefix do domovského priečinka.

PowerShell
# Spusti PowerShell ako správca (Run as Administrator)
PS>npm install -g @anthropic-ai/claude-code
added 47 packages in 11s
# Ak je zakázané spúšťanie skriptov, povol to:
PS>Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
PS>claude --version
claude-code 1.x.x
// windows defender

Windows môže blokovať inštaláciu. V takom prípade spusti PowerShell ako správca a skús znova.

Command Prompt
:: Otvor CMD ako správca (Run as Administrator)
C:\>npm install -g @anthropic-ai/claude-code
added 47 packages in 11s
C:\>claude --version
claude-code 1.x.x
1

VS Code terminál funguje rovnako ako systémový — spusti rovnaký npm príkaz.
3

Po inštalácii reštartuj VS Code, aby sa PATH aktualizoval a príkaz claude bol rozpoznaný.

03

Prihlásenie a autentifikácia

Po inštalácii sa musíš prihlásiť. Claude Code podporuje dva spôsoby autentifikácie: cez prehliadač (Anthropic účet) alebo API kľúčom.

1
Spusti príkaz login
Claude Code otvorí prehliadač automaticky.
2
Prihlás sa na claude.ai
Použi svoj Anthropic / claude.ai účet. Ak nemáš, zaregistruj sa zadarmo.
3
Autorizuj prístup
Klikni na „Authorize" v prehliadači — token sa automaticky uloží do CLI.
prihlásenie
# Prihlásenie cez prehliadač (odporúčané)
$claude login
Opening browser for authentication...
✓ Logged in as michal@example.com
# Alternatíva: API kľúč (pre servery / CI)
$export ANTHROPIC_API_KEY=sk-ant-...
// API kľúč

API kľúč nájdeš na console.anthropic.com → API Keys. Vhodný pre serverové prostredia kde prehliadač nie je dostupný.

04

claude doctor — diagnostika prostredia

Príkaz claude doctor je tvoj prvý kamarát po inštalácii. Skontroluje celé prostredie a oznámi ti, čo funguje a čo treba opraviť.

diagnostika
$claude doctor
✓Node.js version: v22.14.0 (OK)
✓Claude Code version: 1.x.x (latest)
✓Authentication: logged in
✓Network connectivity: OK
⚠Git: not configured (optional)
All systems operational.

Čo doctor kontroluje:

Kontrola Čo overuje Ak zlyhá
Node.js version Minimálna verzia 18+ Aktualizuj Node.js
Claude Code version Porovnáva s najnovšou verziou npm update -g @anthropic-ai/claude-code
Authentication Platnosť tokenu / API kľúča Spusti claude login znova
Network Dostupnosť Anthropic API Skontroluj firewall / proxy
Git Konfigurácia git (voliteľné) Nainštaluj git pre plnú funkcionalitu
05

Prvé spustenie Claude Code

Keď je všetko zelené, spusti Claude Code v ľubovoľnom priečinku projektu. Agent automaticky načíta kontext adresára.

prvé spustenie
# Prejdi do svojho projektu
$cd ~/moj-projekt
# Spusti Claude Code
$claude
Claude Code ready. Working directory: ~/moj-projekt
Model: claude-sonnet-4-6 | Type your message...
> Čo obsahuje tento projekt?
Analyzujem štruktúru projektu...
// rýchly štart

Môžeš zadať úlohu priamo z príkazového riadku: claude "vysvetli mi tento kód" — Claude Code spracuje úlohu a skončí (headless mód).

// rýchly test

Ktorý príkaz overí, že tvoje prostredie je správne nakonfigurované po inštalácii Claude Code?

A claude --version
B npm test
C claude doctor
D claude login --check
// zhrnutie lekcie 02

Claude Code sa inštaluje cez npm install -g @anthropic-ai/claude-code na každej platforme. Po inštalácii spusti claude login a potom claude doctor — ak je všetko zelené, si pripravený pracovať.
