01

Agentic Coding — nová paradigma

Väčšina ľudí pozná AI nástroje ako GitHub Copilot — autocomplete na steroidoch. Napíšeš riadok kódu, AI dokončí zvyšok. To je copilot prístup: model reaktívne reaguje na každý tvoj vstup.

Agentic coding funguje inak. Agent koná — sám navrhne plán, spustí príkazy, skontroluje výsledky, opraví chyby a iteruje, kým úloha nie je hotová. Ty zadáš cieľ, nie každý krok.

🤖 Copilot prístup
  • Reaktívny — čaká na tvoj vstup
  • Dopĺňa kód riadok po riadku
  • Ty riadiš každý krok
  • Kontext = otvorený súbor
  • Žiadne externé akcie
⚡ Agentic prístup
  • Proaktívny — plánuje a koná
  • Zvládne celé featury naraz
  • Agent riadi kroky sám
  • Kontext = celý projekt
  • Spúšťa bash, číta súbory, volá API
💡
Kľúčový insight Agentic coding nenahradí programátora — zosilní ho. Deleguješ rutinné a opakujúce sa úlohy, sám sa sústreďuješ na architektúru a rozhodnutia.

Každý tool call je konkrétna akcia: čítanie súboru, spustenie príkazu v bash, vyhľadávanie v kóde, volanie externého API. Model dostane výsledok späť a rozhodne, čo ďalej.

PRÍKLAD AGENT LOOP
you$ Pridaj jednotkové testy pre funkciu parseDate()
▸ [tool_call] read_file("src/utils/date.ts")
▸ [tool_call] list_files("tests/")
▸ [tool_call] write_file("tests/date.test.ts", ...)
▸ [tool_call] bash("npm test -- date")
✓ 8 testov pridaných, všetky prejdené
ℹ️
Prečo je to dôležité? Agent nepotrebuje tvoj súhlas pri každom kroku. Vie iterovať (napr. opraviť failing test) bez tvojho zásahu. Toto je zásadný rozdiel oproti chatbotom.
03

Claude Code — konkrétne

Claude Code je CLI nástroj (command-line interface) od Anthropic. Beží v tvojom termináli a má prístup k tvojmu projektu, file systému a bashu. Nie je to IDE plugin, nie je to webový chatbot — je to agentic harness postavený okolo modelu Claude.

ZÁKLADNÉ POUŽITIE
# Inštalácia
$ npm install -g @anthropic-ai/claude-code
# Spustenie v priečinku projektu
$ claude
# Jednorázová úloha (headless)
$ claude -p "Zrefaktoruj všetky async funkcie v src/"

Kľúčová výhoda Claude Code Na rozdiel od IDE-first nástrojov, Claude Code je terminál-first. Funguje kdekoľvek — lokálne, na serveri, v CI/CD pipeline, v dev containeri. Maximálna flexibilita.

Zhrnutie lekcie

🔑 Kľúčové pojmy

  • Agentic coding — AI agent koná autonómne, nie len dopĺňa
  • Agent loop — cyklus: prompt → plán → tool call → výsledok → ďalší krok
  • Tool call — konkrétna akcia agenta (bash, read, write, search...)
  • Claude Code — CLI harness od Anthropic, terminál-first, filesystem access
  • Headless mode — spustenie bez interaktívneho rozhrania (claude -p "...")
