s IDE

Claude Code nie je len terminál. Funguje priamo vo VS Code, JetBrains a Chrome — bez prepínania okien, bez straty kontextu.

// čo sa naučíš

  • Nainštalovať a nastaviť Claude Code extension vo VS Code
  • Použiť inline diff — schválenie alebo zamietnutie zmien priamo v editore
  • Nastaviť plugin pre JetBrains IDE (IntelliJ, PyCharm, WebStorm…)
  • Pochopiť, čo prináša Chrome Extension a kedy ju použiť
  • Ovládať klávesové skratky pre rýchlu prácu s agentom
01

Kde všade beží Claude Code

Claude Code má viacero surfaces — rozhraní, cez ktoré s ním môžeš pracovať. Každé má svoje silné stránky a typické použitie.

⌨️
Terminál (CLI)
Základné rozhranie. Plná kontrola, headless mód, CI/CD integrácia.
🖥️
Desktop App
Natívna aplikácia pre macOS a Windows. GUI pre sessions a nastavenia.
🌐
Web (claude.ai)
Prístup odkiaľkoľvek bez inštalácie. Vhodné pre rýchle úlohy.
💙
VS Code Extension
Inline diff, agent panel v editore, priamy prístup k súborom.
🧠
JetBrains Plugin
Pre IntelliJ, PyCharm, WebStorm a ďalšie. Rovnaké funkcie ako VS Code.
🔵
Chrome Extension
Prístup k Claude Code z prehliadača. Toolbar na webových stránkach.
02

VS Code Extension

VS Code extension je najpopulárnejšia integrácia. Pridáva Claude Code panel priamo do editora a umožňuje schvaľovať zmeny kódu inline — bez prepínania do terminálu.

1
Otvor VS Code Extensions
Klávesová skratka: Ctrl+Shift+X (Windows/Linux) alebo Cmd+Shift+X (macOS).
2
Vyhľadaj "Claude Code"
Hľadaj extension od Anthropic — overená modrá fajka. Klikni Install.
3
Prihlás sa
Extension použije existujúci token z CLI. Ak nie si prihlásený, spusti claude login v termináli.
4
Otvor Claude panel
Klikni na ikonu Claude v activity bare (ľavý panel) alebo použi skratku Ctrl+Shift+C.
// alternatíva — príkazový riadok

Ak máš nainštalovaný VS Code CLI: code --install-extension anthropic.claude-code

Inline diff je najcennejšia funkcia VS Code integrácie. Keď agent navrhne zmenu kódu, zobrazí sa priamo v editore — zelené riadky (pridané) a červené (odobrané). Zmenu môžeš schváliť alebo zamietnuť jedným kliknutím.

src/auth/middleware.ts navrhovaná zmena agenta
 export function authMiddleware(req, res, next) {
-  const token = req.headers.token;
+  const token = req.headers.authorization?.split(' ')[1];
   if (!token) {
-    return res.status(401).json({ error: 'No token' });
+    return res.status(401).json({ error: 'Unauthorized' });
   }
 }
workflow s inline diff
# 1. Zadaj úlohu v Claude paneli
> Oprav auth middleware — token by mal byť v Bearer hlavičke
Agent analyzuje kód...
✓ Zmeny navrhnuté — skontroluj diff v editore
# 2. V editore sa zobrazí diff — schváliš alebo zamietneš
[ Schváliť všetky ] [ Schváliť súbor po súbore ] [ Zamietnuť ]
// tip — čiastočné schválenie

Môžeš schváliť zmeny súbor po súbore — nie všetky naraz. Ak s jednou zmenou nesúhlasíš, zamietni ju a požiadaj agenta o úpravu len pre daný súbor.

Akcia Windows / Linux macOS
Otvoriť Claude panel Ctrl+Shift+C Cmd+Shift+C
Nová správa agentovi Ctrl+I Cmd+I
Schváliť diff Ctrl+Enter Cmd+Enter
Zamietnuť diff Ctrl+Backspace Cmd+Backspace
Zastaviť agenta Esc Esc
Pridať aktuálny súbor do kontextu Ctrl+Shift+A Cmd+Shift+A
// vlastné skratky

Klávesové skratky môžeš upraviť v ~/.claude/keybindings.json. Spusti /keybindings-help pre asistenta pri konfigurácii.

03

JetBrains Plugin

Pre používateľov IntelliJ IDEA, PyCharm, WebStorm, GoLand a ďalších JetBrains IDE existuje oficiálny plugin s rovnakými funkciami ako VS Code extension.

1
Otvor Plugin Marketplace
Menu: File → Settings → Plugins (Windows/Linux) alebo IntelliJ IDEA → Preferences → Plugins (macOS).
2
Vyhľadaj "Claude Code"
Hľadaj plugin od Anthropic. Klikni Install a reštartuj IDE.
3
Otvor Claude tool window
Po reštarte nájdeš Claude panel v pravom alebo dolnom tool window. Prípadne cez menu: View → Tool Windows → Claude.
4
Autentifikácia
Plugin zdieľa token s CLI — ak si prihlásený cez claude login, plugin funguje automaticky.
// rovnaké funkcie

JetBrains plugin má rovnaký inline diff, schvaľovanie zmien aj klávesové skratky ako VS Code. Skratky sú mierne odlišné — plugin ich zobrazí pri prvom spustení.

04

Chrome Extension

Chrome Extension prináša Claude Code priamo do prehliadača. Je užitočná najmä pri práci s webovými aplikáciami — agent vidí aktuálnu stránku a môže reagovať na jej obsah.

👁️
Toolbar na stránke
Zobrazí Claude panel priamo na webovej stránke bez prepínania kontextu.
📸
Screenshot kontext
Agent vidí aktuálnu stránku — môžeš sa pýtať na UI, chyby alebo obsah.
💬
Toolbar komentáre
Pridávaj komentáre a feedbacky priamo na stránku — zdieľaj s tímom.
🔗
Prepojenie s CLI
Extension je prepojená s terminálovým Claude Code — zdieľajú autentifikáciu.
typický use-case — Chrome Extension
# Vidíš bug na stránke? Klikni na Claude toolbar icon
> Táto stránka má bug — login form sa neodošle. Analyzuj.
Agent vidí screenshot stránky...
✓ Identifikovaný problém: chýba event.preventDefault() v submit handleri
# Agent môže priamo otvoriť súbor v editore a opraviť ho
// kedy Chrome Extension použiť

Chrome Extension je ideálna pre vizuálny debugging a UI review. Pre samotné kódovanie je terminál alebo IDE extension efektívnejší.

05

Ktoré rozhranie použiť kedy

prehľad použitia
CLI (terminál)
→ Headless úlohy, CI/CD, scripty, serverové prostredia
→ Plná kontrola, žiadne GUI overhead
VS Code / JetBrains
→ Každodenné kódovanie, refaktoring, code review
→ Inline diff — vidíš zmeny priamo v kontexte kódu
Chrome Extension
→ UI debugging, vizuálna spätná väzba, stránkový kontext
→ Toolbar komentáre pre tím
Web / Desktop App
→ Rýchle konzultácie, práca bez lokálnej inštalácie
→ GUI pre správu sessions a nastavení

B Inline diff — zobrazí zmeny priamo v editore, schváliš súbor po súbore
C Chrome Extension — vizuálny náhľad zmien
D git diff — pozrieš si zmeny po commite
// zhrnutie lekcie 06

Claude Code beží na 6 surfaces — CLI, Desktop, Web, VS Code, JetBrains a Chrome. Pre každodennú prácu je VS Code alebo JetBrains ideálny vďaka inline diff. CLI zostáva základom pre automatizáciu a CI/CD. Chrome Extension sa hodí pri UI debuggingu.
